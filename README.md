# LOGATE – Logic Gate Simulator

LOGATE is a web-based Logic Gate Simulator developed to help students understand digital electronics in an easy and interactive way.

## Live Demo
🔗 (https://mikacom.github.io/LOGATE/)

## Features

- Welcome page with modern UI
- Secure login page using Firebase Authentication
- Drag and drop logic gates
- Supports AND, OR, NOT, XOR gates
- Connect gates using wires
- Toggle Input A and B
- Real-time output display
- Attractive glassmorphism interface

## Technologies Used

- HTML
- CSS
- JavaScript
- Firebase Authentication
- GitHub Pages

## How to Use

1. Open the website link  
2. Click Continue  
3. Login using registered email and password  
4. Enter simulator page  
5. Drag logic gates into workspace  
6. Connect gates with wires  
7. Toggle inputs A and B  
8. View output result instantly

## Educational Purpose

This project is useful for students learning:

- Digital Electronics
- Logic Gates
- Boolean Algebra
- Basic Circuit Design
